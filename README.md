### 사용 라이브러리

- [Create React App](https://github.com/facebook/create-react-app). 기본 세팅 라이브러리
- react-router-dom
- axios
- http-proxy-middleware (api 요청시 cors 문제 해결 위함)
- 디자인이 많이 필요해 보이지 않아 스타일 라이브러리 없이 css만으로 스타일링.
